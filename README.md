# Telegram-бот, контейнеризованный с помощью Docker

## Краткое описание проекта и цель работы
Данный проект представляет собой простой Telegram-бот, разработанный на языке Python и развёртываемый в Docker-контейнере.  
Бот принимает текстовые сообщения от пользователя и отправляет ответное сообщение.

Целью проекта является:
- изучение основ разработки Telegram-ботов;
- освоение принципов контейнеризации приложений с помощью Docker;
- формирование навыков корректного оформления программных проектов.

---

## Архитектура и структура проекта
Проект имеет простую и наглядную структуру:

telegram_bot_docker/
bot.py # основной файл с кодом Telegram-бота
Dockerfile # инструкция для сборки Docker-образа
requirements.txt # зависимости Python
README.md # описание проекта

Архитектура проекта является монолитной и предназначена для учебных целей.

## Запуск проекта

## Запуск локально (без Docker)
Для запуска локально необходимо установленное окружение Python 3.

1. Установить зависимости:
```bash
pip install -r requirements.txt
2. Установить переменную окружения с токеном бота:
export BOT_TOKEN=ВАШ_ТОКЕН
3. Запустить бота:
python bot.py
Запуск в Docker
Сборка Docker-образа
docker build -t tg-bot .
Запуск контейнера
docker run --rm -e BOT_TOKEN=ВАШ_ТОКЕН tg-bot
Конфигурация

Проект использует переменные окружения для конфигурации.

Переменная	Описание
BOT_TOKEN	Токен Telegram-бота, получаемый через BotFather

Примеры использования
Входные данные

текстовое сообщение, отправленное пользователем в Telegram-чат с ботом.

Выходные данные

ответное сообщение бота с повторением введённого текста.
