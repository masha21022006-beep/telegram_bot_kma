# Telegram-бот в Docker

Данная курсовая работа посвящена разработке простого Telegram-бота,
запускаемого в контейнере Docker.

## Цель работы
Освоение принципов контейнеризации приложений и разработки
Telegram-ботов на языке Python.

## Используемые технологии
- Python 3
- Docker
- Telegram Bot API

## Структура проекта
- bot.py — исходный код бота
- Dockerfile — описание Docker-контейнера
- requirements.txt — зависимости Python

## Запуск проекта
1. Собрать Docker-образ:
   docker build -t tg-bot .

2. Запустить контейнер:
   docker run -e BOT_TOKEN=ВАШ_ТОКЕН tg-bot

## Результат
В результате работы создан Telegram-бот, который принимает сообщения
от пользователя и отвечает на них.
